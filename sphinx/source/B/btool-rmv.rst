BTool\_RMV
==========

+----------+-------------------------------------------------------------------+
| Syntax   |  BTool\_RMV                                                       |
+----------+-------------------------------------------------------------------+
| Location |  BTool                                                            |
+----------+-------------------------------------------------------------------+

All keywords implemented by BTool (except BTool\_EXT) are removed from
the SuperBASIC name list. The Toolkit itself remains in memory and can
be re-activated with BTool\_EXT.
