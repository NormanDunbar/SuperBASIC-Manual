BVER$
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  BVER$                                                            |
+----------+-------------------------------------------------------------------+
| Location |  BeuleTools                                                       |
+----------+-------------------------------------------------------------------+

This function returns the version number of the Beule Toolkit. This may
be useful if a program makes use of commands or functions which were not
supported by older versions.
