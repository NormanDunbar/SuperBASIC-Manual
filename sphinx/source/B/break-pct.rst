BREAK%
======

+----------+-------------------------------------------------------------------+
| Syntax   |  BREAK%                                                           |
+----------+-------------------------------------------------------------------+
| Location |  BTool                                                            |
+----------+-------------------------------------------------------------------+

The function BREAK% returns the current state as to whether the Break
key is enabled, either ON or OFF.
