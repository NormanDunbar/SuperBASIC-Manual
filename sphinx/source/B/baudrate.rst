..  _baudrate:

BAUDRATE
========

+----------+-------------------------------------------------------------------+
| Syntax   |  BAUDRATE                                                         |
+----------+-------------------------------------------------------------------+
| Location |  SERMouse                                                         |
+----------+-------------------------------------------------------------------+

This function returns the actual baud rate of the system which will be
used on any newly opened serial port channel.


**CROSS-REFERENCE**

The system's baud rate is set with :ref:`baud`.
