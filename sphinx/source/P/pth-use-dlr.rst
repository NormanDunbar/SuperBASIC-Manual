PTH\_USE$
=========

+----------+-------------------------------------------------------------------+
| Syntax   |  PTH\_USE$                                                        |
+----------+-------------------------------------------------------------------+
| Location |  Path device                                                      |
+----------+-------------------------------------------------------------------+

As mentioned above, the function PTH\_USE$ gives you the name which is
used for the path device.

**Example**

::

    PRINT PTH_USE$
