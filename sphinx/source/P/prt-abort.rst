PRT\_ABORT
==========

+----------+-------------------------------------------------------------------+
| Syntax   |  PRT\_ABORT                                                       |
+----------+-------------------------------------------------------------------+
| Location |  ST/QL, SMSQ/E                                                    |
+----------+-------------------------------------------------------------------+

This is the same as PAR\_ABORT or SER\_ABORT, depending on which device
PRT is linked to.

**CROSS-REFERENCE**

See :ref:`ser-abort` and
:ref:`par-abort`.
:ref:`prt-use` allows you to specify which port prt emulates.
