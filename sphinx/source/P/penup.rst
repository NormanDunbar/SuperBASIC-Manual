PENUP
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  PENUP [#ch]                                                      |
+----------+-------------------------------------------------------------------+
| Location |  QL ROM                                                           |
+----------+-------------------------------------------------------------------+

This command places the turtle's pen to the up position in the
specified window (default #1), thus preventing any further drawing.

See :ref:`pendown` and
:ref:`move` for more details.
