PAR\_GETFILTER
==============

+----------+-------------------------------------------------------------------+
| Syntax   | state% = PAR\_GETFILTER(port%)                                    |
+----------+-------------------------------------------------------------------+
| Location | SMSQ/E for QPC                                                    |
+----------+-------------------------------------------------------------------+

This returns whether the printer filter is enabled for the specified port.
