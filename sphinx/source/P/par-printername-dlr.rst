PAR\_PRINTERNAME$
=================

+----------+-------------------------------------------------------------------+
| Syntax   | name$ = PAR\_PRINTERNAME$(n)                                      |
+----------+-------------------------------------------------------------------+
| Location | SMSQ/E for QPC                                                    |
+----------+-------------------------------------------------------------------+

This returns the name of printer number n (counted from 1 to :ref:`par-printercount`\ ).
