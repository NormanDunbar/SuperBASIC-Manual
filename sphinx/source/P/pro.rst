..  _pro:

PRO
===

+----------+-------------------------------------------------------------------+
| Syntax   |  PRO                                                              |
+----------+-------------------------------------------------------------------+
| Location |  Beuletools                                                       |
+----------+-------------------------------------------------------------------+

This function returns the codes needed to switch on the proportional
font on an EPSON compatible printer::

    PRINT PRO

is the same as::

    PRINT CHR$(27) & "p" & CHR$(1)
