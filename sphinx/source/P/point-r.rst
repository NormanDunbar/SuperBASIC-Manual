POINT\_R
========

+----------+-------------------------------------------------------------------------+
| Syntax   |  POINT\_R [#ch,] x,y :sup:`\*`\ [;x\ :sup:`i`\ ,y\ :sup:`i`]\ :sup:`\*` |
+----------+-------------------------------------------------------------------------+
| Location |  QL ROM                                                                 |
+----------+-------------------------------------------------------------------------+

This command is similar to POINT except that all co-ordinates given are
relative to the current graphics pointer.
