PROCedure
=========

+----------+-------------------------------------------------------------------+
| Syntax   |  ... PROCedure                                                    |
+----------+-------------------------------------------------------------------+
| Location |  QL ROM                                                           |
+----------+-------------------------------------------------------------------+

This keyword forms part of the structure DEFine PROCedure. As such, it
cannot be used on its own within a program - this will cause a 'bad
line' error.
