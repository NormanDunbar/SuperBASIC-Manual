PTH\_LIST
=========

+----------+-------------------------------------------------------------------+
| Syntax   |  PTH\_LIST [#ch]                                                  |
+----------+-------------------------------------------------------------------+
| Location |  Path device                                                      |
+----------+-------------------------------------------------------------------+

The command PTH\_LIST prints a list of the search paths available to
the PTH device.

**Examples**

::

    PTH_LIST
    PTH_LIST#2

**CROSS-REFERENCE**

:ref:`pth-dlr` is an alternative way to get the path
list.
