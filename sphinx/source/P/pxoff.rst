PXOFF
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  PXOFF                                                            |
+----------+-------------------------------------------------------------------+
| Location |  PEX                                                              |
+----------+-------------------------------------------------------------------+

This command is the same as PIE\_EX\_OFF except for the PEX system
extension - it disables PEX for SD.EXTOP system calls, so that they are
trapped by the Pointer Environment.

**CROSS-REFERENCE**

See :ref:`pxon` and\ :ref:`peon`.
