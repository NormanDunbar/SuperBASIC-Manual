PXON
====

+----------+-------------------------------------------------------------------+
| Syntax   |  PXON                                                             |
+----------+-------------------------------------------------------------------+
| Location |  PEX                                                              |
+----------+-------------------------------------------------------------------+

This command is the same as PIE\_EX\_ON except for the PEX system
extension - it enables PEX for SD.EXTOP system calls, so that they can
work in the background.

**CROSS-REFERENCE**

and\ :ref:`peon`.
