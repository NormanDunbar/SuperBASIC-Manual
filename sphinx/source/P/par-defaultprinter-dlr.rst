PAR\_DEFAULTPRINTER$
====================

+----------+-------------------------------------------------------------------+
| Syntax   | name$ = PAR\_DEFAULTPRINTER$                                      |
+----------+-------------------------------------------------------------------+
| Location | SMSQ/E for QPC                                                    |
+----------+-------------------------------------------------------------------+

This returns the name of Windows' default printer. The name can later be used with :ref:`par-setprinter` for example.
