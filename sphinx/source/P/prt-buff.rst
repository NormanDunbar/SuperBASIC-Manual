PRT\_BUFF
=========

+----------+-------------------------------------------------------------------+
| Syntax   |  PRT\_BUFF [size]                                                 |
+----------+-------------------------------------------------------------------+
| Location |  ST/QL, SMSQ/E                                                    |
+----------+-------------------------------------------------------------------+

This is exactly the same as PAR\_BUFF except that it creates buffered
output on whichever port is attached to the PRT device.
