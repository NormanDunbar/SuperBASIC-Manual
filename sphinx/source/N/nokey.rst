NOKEY
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  NOKEY                                                            |
+----------+-------------------------------------------------------------------+
| Location |  KEYMAN                                                           |
+----------+-------------------------------------------------------------------+

This command temporarily disables all definitions of KEY. Pressing the
key combination <CTRL><CAPS> toggles between enabled and disabled state,
so this is partially equivalent to NOKEY.
