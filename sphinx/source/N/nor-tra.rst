NOR\_TRA
========

+----------+-------------------------------------------------------------------+
| Syntax   |  NOR\_TRA                                                         |
+----------+-------------------------------------------------------------------+
| Location |  ST/QL                                                            |
+----------+-------------------------------------------------------------------+

This is the complementary function to NOR\_MSG and points to the
printer translation table for Norway contained in the file
NOR\_TRA\_rext.
