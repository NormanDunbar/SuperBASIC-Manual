..  _mkl-dlr:

MKL$
====

+----------+-------------------------------------------------------------------+
| Syntax   |  MKL$ (longint) where longint = -2\*INTMAX-1..2\*INTMAX+1         |
+----------+-------------------------------------------------------------------+
| Location |  BTool                                                            |
+----------+-------------------------------------------------------------------+

This function returns a string containing the internal format of a long
integer number (which is stored as four bytes).
