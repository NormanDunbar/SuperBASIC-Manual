MNET\_OFF
=========

+----------+-------------------------------------------------------------------+
| Syntax   |  MNET\_OFF                                                        |
+----------+-------------------------------------------------------------------+
| Location |  SMSQ/E, ATARI Emulators                                          |
+----------+-------------------------------------------------------------------+

This command turns the MIDINET driver off temporarily so that you can
use the MIDI ports independently.

See :ref:`mnet-on`.
