MKF$
====

+----------+-------------------------------------------------------------------+
| Syntax   |  MKF$ (float)                                                     |
+----------+-------------------------------------------------------------------+
| Location |  BTool                                                            |
+----------+-------------------------------------------------------------------+

This function returns a string containing the internal representation
of a floating point number (which is stored as six bytes).
