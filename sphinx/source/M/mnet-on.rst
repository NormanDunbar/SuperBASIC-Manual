..  _mnet-on:

MNET\_ON
========

+----------+-------------------------------------------------------------------+
| Syntax   |  MNET\_ON                                                         |
+----------+-------------------------------------------------------------------+
| Location |  SMSQ/E, ATARI Emulators                                          |
+----------+-------------------------------------------------------------------+

This command switches the MIDINET driver back on after it has been
disabled with MNET\_OFF.

**CROSS-REFERENCE**

See :ref:`mnet-off`. Also see
:ref:`midinet`.
