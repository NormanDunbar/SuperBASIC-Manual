MNET%
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  MNET%                                                            |
+----------+-------------------------------------------------------------------+
| Location |  SMSQ/E, ATARI Emulators                                          |
+----------+-------------------------------------------------------------------+

This function returns the current station number of the computer as set
with MNET.
