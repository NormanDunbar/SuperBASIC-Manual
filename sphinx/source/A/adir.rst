..  _adir:

ADIR
====

+----------+-------------------------------------------------------------------+
| Syntax   |  ADIR [#channel,] device                                          |
+----------+-------------------------------------------------------------------+
| Location |  ATARIDOS                                                         |
+----------+-------------------------------------------------------------------+

This command is the same as DIR except that it works on ATARI disks or
IBM PS/2 Disks.
