ARCOTH
======

+----------+-------------------------------------------------------------------+
| Syntax   |  ARCOTH (x)                                                       |
+----------+-------------------------------------------------------------------+
| Location |  Hyper                                                            |
+----------+-------------------------------------------------------------------+

This function returns the arc hyperbolic cotangent of the specified
value ie.

ARCOTH ( COTH ( x )) = x

Or to keep it simple, it can be returned with the equivalent expression
LN((x+1) / (x-1)) / 2

**CROSS-REFERENCE**

See :ref:`acot`, :ref:`arcosh`,
and :ref:`artanh`.
