ARSINH
======

+----------+-------------------------------------------------------------------+
| Syntax   |  ARSINH (x)                                                       |
+----------+-------------------------------------------------------------------+
| Location |  Hyper                                                            |
+----------+-------------------------------------------------------------------+

This function is the arc hyperbolic sine (ie. the complementary function
to SINH).

The SuperBASIC expression:

LN(x + SQRT(x\*x-1))

gives the same value.

**CROSS-REFERENCE**

See :ref:`asin`, :ref:`arcosh`,
and :ref:`arcoth`.
