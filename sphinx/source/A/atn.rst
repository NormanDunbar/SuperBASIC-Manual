..  _atn:

ATN
===

+----------+-------------------------------------------------------------------+
| Syntax   |  ATN (x)                                                          |
+----------+-------------------------------------------------------------------+
| Location |  Math Package                                                     |
+----------+-------------------------------------------------------------------+

This function is the same as the original QL ROM variant of ATAN.

**NOTE**

ATN has been implemented to make porting programs written in other BASIC
dialects easier.

**CROSS-REFERENCE**

See :ref:`atan`.
