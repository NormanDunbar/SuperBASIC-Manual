..  _resave:

RESAVE
======

+----------+-------------------------------------------------------------------+
| Syntax   |  RESAVE program\_name                                             |
+----------+-------------------------------------------------------------------+
| Location |  MutiBASIC (DIY Toolkit - Vol M)                                  |
+----------+-------------------------------------------------------------------+

This command is the same as UNLOAD except that if the specified
program\_name has already been stored in memory, it is overwritten.

**CROSS-REFERENCE**

See :ref:`unload`!
