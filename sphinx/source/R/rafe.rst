RAFE
====

+----------+-------------------------------------------------------------------+
| Syntax   |  RAFE (i,n)                                                       |
+----------+-------------------------------------------------------------------+
| Location |  Toolfin                                                          |
+----------+-------------------------------------------------------------------+

The function RAFE returns the value of (((1+i)\ :sup:`(1/n-1)` \* n )/i
where i and n can be any floating point numbers (see MT for error
handling).
