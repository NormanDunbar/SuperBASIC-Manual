..  _rtp-t:

RTP\_T
======

+----------+-------------------------------------------------------------------+
| Syntax   |  RTP\_T (imag, real)                                              |
+----------+-------------------------------------------------------------------+
| Location |  PTRRTP                                                           |
+----------+-------------------------------------------------------------------+

The function RTP\_T takes rectangular co-ordinates and returns the
corresponding argument, (the angle used in polar co-ordinates) in
radians. See RTP\_R for further information.
