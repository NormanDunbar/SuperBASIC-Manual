RAE
===

+----------+-------------------------------------------------------------------+
| Syntax   |  RAE (i,n)                                                        |
+----------+-------------------------------------------------------------------+
| Location |  Toolfin                                                          |
+----------+-------------------------------------------------------------------+

The function RAE returns the value of: i/(((1+i)\ :sup:`1/n`\ ) -1)
\*n )
where i and n can be any floating point numbers (see MT for error
handling).
