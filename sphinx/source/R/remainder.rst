REMAINDER
=========

+----------+-------------------------------------------------------------------+
| Syntax   |  REMAINDER                                                        |
+----------+-------------------------------------------------------------------+
| Location |  QL ROM                                                           |
+----------+-------------------------------------------------------------------+

This keyword can only be used within a SELect ON structure. It is used
to represent all possible untested values of the SELect ON variable.

**CROSS-REFERENCE**

Please see :ref:`select--on`.
