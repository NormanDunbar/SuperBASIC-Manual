REV$
====

+----------+-------------------------------------------------------------------+
| Syntax   |  REV$ (string$)                                                   |
+----------+-------------------------------------------------------------------+
| Location |  REV                                                              |
+----------+-------------------------------------------------------------------+

This function returns the supplied string in reverse order.

**Example**

::

    PRINT REV$("Hello World")

shows dlroW olleH
