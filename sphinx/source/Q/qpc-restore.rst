QPC\_RESTORE
============

+----------+-------------------------------------------------------------------+
| Syntax   | QPC\_RESTORE                                                      |
+----------+-------------------------------------------------------------------+
| Location | SMSQ/E for QPC                                                    |
+----------+-------------------------------------------------------------------+

Restores the QPC window. This will return the window size from minimised or maximised to what it was before.
