QPC\_HOSTOS
===========

+----------+-------------------------------------------------------------------+
| Syntax   | os% = QPC\_HOSTOS                                                 |
+----------+-------------------------------------------------------------------+
| Location | SMSQ/E for QPC                                                    |
+----------+-------------------------------------------------------------------+

This function returns the host operating system under which QPC was started.

Possible return codes are:

- 0 = DOS (QPC1)
- 1 = Win9x/ME (QPC2)
- 2 = WinNT/2000/XP (QPC2)
