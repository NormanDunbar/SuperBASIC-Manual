QPC\_VER$
=========

+----------+-------------------------------------------------------------------+
| Syntax   | v$ = QPC\_VER$                                                    |
+----------+-------------------------------------------------------------------+
| Location | SMSQ/E for QPC                                                    |
+----------+-------------------------------------------------------------------+

This returns the current QPC version.

**Example**

::

    PRINT QPC_VER$

Will print 4.00 or higher.
