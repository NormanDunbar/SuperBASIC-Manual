QLRUN
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  QLRUN [device\_]filename                                         |
+----------+-------------------------------------------------------------------+
| Location |  SMS                                                              |
+----------+-------------------------------------------------------------------+

This command is exactly the same as QLOAD except that the program is
automatically RUN as soon as it has been loaded into memory.

See :ref:`qload` and
