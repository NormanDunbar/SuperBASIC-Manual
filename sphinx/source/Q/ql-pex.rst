QL\_PEX
=======

+----------+-------------------------------------------------------------------+
| Syntax   |  QL\_PEX                                                          |
+----------+-------------------------------------------------------------------+
| Location |  PEX                                                              |
+----------+-------------------------------------------------------------------+

This function returns the offset of the keyword linkage block of the
keywords added by the PEX toolkit. This offset is needed for
Qliberator's $$asmb directorive.

See :ref:`pex-save`\ .
