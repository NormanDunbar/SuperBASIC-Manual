..  _off:

OFF
===

+----------+-------------------------------------------------------------------+
| Syntax   |  OFF                                                              |
+----------+-------------------------------------------------------------------+
| Location |  BTool                                                            |
+----------+-------------------------------------------------------------------+

This is a constant which returns 0. OFF and ON are intended to make
listings more readable.

**Example**

::

    FREEZE OFF
