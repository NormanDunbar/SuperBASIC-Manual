..  _dbl:

DBL
===

+----------+-------------------------------------------------------------------+
| Syntax   |  DBL                                                              |
+----------+-------------------------------------------------------------------+
| Location |  Beuletools                                                       |
+----------+-------------------------------------------------------------------+

This function returns the control codes needed to switch on emphasised mode on an EPSON compatible printer: DBL=CHR$(27)&"E".
