GER\_TRA
========

+----------+-------------------------------------------------------------------+
| Syntax   |  GER\_TRA                                                         |
+----------+-------------------------------------------------------------------+
| Location |  ST/QL                                                            |
+----------+-------------------------------------------------------------------+

This is the complementary function to GER\_MSG and points to the
printer translation table for Germany contained in the file
GER\_TRA\_rext.
