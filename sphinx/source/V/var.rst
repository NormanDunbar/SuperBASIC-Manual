..  _var:

VAR
===

+----------+-------------------------------------------------------------------+
| Syntax   |  VAR (i,n)                                                        |
+----------+-------------------------------------------------------------------+
| Location |  Toolfin                                                          |
+----------+-------------------------------------------------------------------+

The function VAR returns the value of: ((1+i)\ :sup:`n`-1) / (i\*
(1+i)\ :sup:`n`) where i and n can be any floating point numbers (see MT for error
handling).
