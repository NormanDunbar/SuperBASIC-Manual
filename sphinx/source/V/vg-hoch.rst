VG\_HOCH
========

+----------+-------------------------------------------------------------------+
| Syntax   |  VG\_HOCH (fontnr) fontnr=0..15                                   |
+----------+-------------------------------------------------------------------+
| Location |  BGI                                                              |
+----------+-------------------------------------------------------------------+

This function returns the maximum height of the specified font
(fontnr=0..15) in pixels if printed with the current size settings of
VG\_PARA.

**CROSS-REFERENCE**

:ref:`vg-para` and
:ref:`vg-load`.
