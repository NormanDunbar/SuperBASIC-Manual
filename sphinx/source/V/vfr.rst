..  _vfr:

VFR
===

+----------+-------------------------------------------------------------------+
| Syntax   |  VFR (i,n)                                                        |
+----------+-------------------------------------------------------------------+
| Location |  Toolfin                                                          |
+----------+-------------------------------------------------------------------+

The function VFR returns the value of: ((1+i)\ :sup:`n`-1)/i
where i and n can be any floating point numbers (see MT for error
handling).
