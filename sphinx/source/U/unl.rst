..  _unl:

UNL
===

+----------+-------------------------------------------------------------------+
| Syntax   |  UNL                                                              |
+----------+-------------------------------------------------------------------+
| Location |  Beuletools                                                       |
+----------+-------------------------------------------------------------------+

This function returns the control codes needed to switch on underline
printing on an EPSON compatible printer, PRINT UNL is the same as::

    PRINT CHR$(27) & "-" & CHR$(1)
