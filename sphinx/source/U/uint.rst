UINT
====

+----------+-------------------------------------------------------------------+
| Syntax   |  UINT (x%)                                                        |
+----------+-------------------------------------------------------------------+
| Location |  BTool                                                            |
+----------+-------------------------------------------------------------------+

The function UINT returns the unsigned value of a (signed) integer::

    unsigned = signed% + 2^16

or::

    unsigned = UINT(signed%)
