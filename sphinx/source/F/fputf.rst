FPUTF
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  FPUTF [#ch,] float :sup:`\*`\ [,float\ :sup:`i`]\ :sup:`\*`      |
+----------+-------------------------------------------------------------------+
| Location |  BTool                                                            |
+----------+-------------------------------------------------------------------+

This command writes the floating point number(s) float in its
internal format (six bytes) to #ch, default is #1.
