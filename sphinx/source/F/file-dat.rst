FILE\_DAT
=========

+----------+------------------------------------------------------------------+
| Syntax   | FILE\_DAT (filename) or                                          |
|          |                                                                  |
|          | FILE\_DAT (file$)                                                |
+----------+------------------------------------------------------------------+
| Location | TinyToolkit                                                      |
+----------+------------------------------------------------------------------+

This is the same as FDAT except that default devices and sub-
directories are not supported.
