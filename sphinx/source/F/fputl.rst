FPUTL
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  FPUTL [#ch,] longint :sup:`\*`\ [,longint\ :sup:`i`]\ :sup:`\*`  |
+----------+-------------------------------------------------------------------+
| Location |  BTool                                                            |
+----------+-------------------------------------------------------------------+

This command writes the specified long integer(s) longint
(-231..231-2) in internal format (four bytes) to #ch, default is #1.
