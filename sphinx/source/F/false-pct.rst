FALSE%
======

+----------+-------------------------------------------------------------------+
| Syntax   |  FALSE%                                                           |
+----------+-------------------------------------------------------------------+
| Location |  TRUFA                                                            |
+----------+-------------------------------------------------------------------+

The function FALSE% returns the constant 0. It is used to write
programs which are more legible or which adopt habits from the PASCAL
language.
