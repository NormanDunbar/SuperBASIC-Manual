FPUT%
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  FPUT% [#ch,] integer :sup:`\*`\ [,integer\ :sup:`i`]\ :sup:`\*`  |
+----------+-------------------------------------------------------------------+
| Location |  BTool                                                            |
+----------+-------------------------------------------------------------------+

This command writes the specified integer(s) (range 0...32767) in its
internal format to #ch, default is #1. An integer is stored internally
as two bytes (one word).
