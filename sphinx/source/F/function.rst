FuNction
========

+----------+-------------------------------------------------------------------+
| Syntax   |  ... FuNction                                                     |
+----------+-------------------------------------------------------------------+
| Location |  QL ROM                                                           |
+----------+-------------------------------------------------------------------+

This keyword forms part of the structure DEFine FuNction. As such, it
cannot be used on its own within a program - this will cause a 'bad
line' error.
