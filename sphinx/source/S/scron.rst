SCRON
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  SCRON                                                            |
+----------+-------------------------------------------------------------------+
| Location |  Ecran Manager                                                    |
+----------+-------------------------------------------------------------------+

The SCRON command makes the screen visible once again after it has been
disabled with SCROF.

**Example**

::

    SCRON

**NOTE**

See SSAVE.
