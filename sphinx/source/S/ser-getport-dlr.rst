SER\_GETPORT$
=============

+----------+-------------------------------------------------------------------+
| Syntax   | com$ = SER\_GETPORT$(port%)                                       |
+----------+-------------------------------------------------------------------+
| Location | SMSQ/E for QPC                                                    |
+----------+-------------------------------------------------------------------+

Returns the device the SER port is connected to, for example "COM1".

See :ref:`ser-setport`.
