SELect
======

+----------+-------------------------------------------------------------------+
| Syntax   |  SELect                                                           |
+----------+-------------------------------------------------------------------+
| Location |  QL ROM                                                           |
+----------+-------------------------------------------------------------------+

This keyword forms an integral part of the SELect ON structure
identifier and has no use on its own. If you try to enter it on its own,
the error 'Bad Name' will be generated.
