..  _si:

SI
==

+----------+-------------------------------------------------------------------+
| Syntax   |  SI                                                               |
+----------+-------------------------------------------------------------------+
| Location |  Beuletools                                                       |
+----------+-------------------------------------------------------------------+

This function contains the control codes needed to switch on condensed
print on an EPSON compatible printer::

    PRINT SI

is the same as::

    PRINT CHR$(15)
