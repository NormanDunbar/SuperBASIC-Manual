ENV\_LIST
=========

+----------+-------------------------------------------------------------------+
| Syntax   |  ENV\_LIST [#ch]                                                  |
+----------+-------------------------------------------------------------------+
| Location |  Environment Variables                                            |
+----------+-------------------------------------------------------------------+

This command lists all currently active environment variables to the specified
channel (default #1).

**CROSS-REFERENCE**

Please see SETENV.
