END
===

+----------+-------------------------------------------------------------------+
| Syntax   |  END ...                                                          |
+----------+-------------------------------------------------------------------+
| Location |  QL ROM                                                           |
+----------+-------------------------------------------------------------------+

This keyword forms part of the structures: END WHEN, END SELect, END IF, END REPeat,
END FOR and END DEFine  As such, it cannot be used on its own within a
program - this will cause a 'bad line' error.
