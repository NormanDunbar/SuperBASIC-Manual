ELSE
====

+----------+-------------------------------------------------------------------+
| Syntax   |  ELSE :sup:`\*`\ [:statements]\ :sup:`\*`                         |
+----------+-------------------------------------------------------------------+
| Location |  QL ROM                                                           |
+----------+-------------------------------------------------------------------+

This command forms part of the IF...END IF structure and allows you to take alternative
action if the condition contained in the IF statement proves to be false.
