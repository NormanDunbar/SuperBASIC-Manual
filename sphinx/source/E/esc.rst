..  _esc:

ESC
===

+----------+-------------------------------------------------------------------+
| Syntax   |  ESC                                                              |
+----------+-------------------------------------------------------------------+
| Location |  Beuletools                                                       |
+----------+-------------------------------------------------------------------+

This function returns the control codes ESC, or CHR$(27) for use on an EPSON compatible printer::

    PRINT ESC

is the same as::

    PRINT CHR$(27)
