T\_STOP
=======

+----------+-------------------------------------------------------------------+
| Syntax   |  T\_STOP [watch]                                                  |
+----------+-------------------------------------------------------------------+
| Location |  Timings (DIY Toolkit - Vol H)                                    |
+----------+-------------------------------------------------------------------+

This command stops the specified stop-watch (default 1) from running.

**CROSS-REFERENCE**

:ref:`t-restart` restarts a stop-watch.
:ref:`t-start` starts a stop-watch from afresh.
