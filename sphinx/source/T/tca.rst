..  _tca:

TCA
===

+----------+-------------------------------------------------------------------+
| Syntax   |  TCA (i,n)                                                        |
+----------+-------------------------------------------------------------------+
| Location |  Toolfin                                                          |
+----------+-------------------------------------------------------------------+

The function TCA returns the value of: i/(1-(1+i)\ :sup:`-n`)
where i and n can be any floating point numbers (see MT for error
handling).
