TK\_VER$
========

+----------+-------------------------------------------------------------------+
| Syntax   |  TK\_VER$                                                         |
+----------+-------------------------------------------------------------------+
| Location |  Turbo Toolkit                                                    |
+----------+-------------------------------------------------------------------+

This function returns the version ID of the Turbo Toolkit, eg. 3e27

**NOTE**

Before v3.00 the Turbo Toolkit did not install properly under Minerva
and SMS.
