..  _troff:

TROFF
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  TROFF                                                            |
+----------+-------------------------------------------------------------------+
| Location |  Minerva (TRACE)                                                  |
+----------+-------------------------------------------------------------------+

This command turns off the trace function and closes any file
associated with the trace output.

**CROSS-REFERENCE**

:ref:`tron` and :ref:`sstep` turn
the trace function on.
