TNC
===

+----------+-------------------------------------------------------------------+
| Syntax   |  TNC (i,n)                                                        |
+----------+-------------------------------------------------------------------+
| Location |  Toolfin                                                          |
+----------+-------------------------------------------------------------------+

The function TNC returns the value of: n\*((1+i)\ :sup:`1/n`-1)
where i and n can be any floating point numbers (see MT for error
handling).
