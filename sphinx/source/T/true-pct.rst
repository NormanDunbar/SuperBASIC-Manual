TRUE%
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  TRUE%                                                            |
+----------+-------------------------------------------------------------------+
| Location |  TRUFA                                                            |
+----------+-------------------------------------------------------------------+

TRUE% is the constant 1. It is used to write programs which are more
legible or which adopt habits from the PASCAL language.

**Example**

::

    IF QuATARI=TRUE% THEN ...

is the same as::

    IF QuATARI THEN ...
