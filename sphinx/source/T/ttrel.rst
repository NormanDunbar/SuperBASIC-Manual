TTREL
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  TTREL adr                                                        |
+----------+-------------------------------------------------------------------+
| Location |  QView Tiny Toolkit                                               |
+----------+-------------------------------------------------------------------+

This is similar to the RECHP command, except that it will only remove
areas set aside with TTALL.
