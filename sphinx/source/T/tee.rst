..  _tee:

TEE
===

+----------+-------------------------------------------------------------------+
| Syntax   |  TEE (i,n)                                                        |
+----------+-------------------------------------------------------------------+
| Location |  Toolfin                                                          |
+----------+-------------------------------------------------------------------+

The function TEE returns the value of (1+(i/n)\ :sup:`n`-1)
where i and n can be any floating point numbers (see MT for error
handling).
