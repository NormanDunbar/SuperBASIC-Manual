LRESFAST
========

+----------+-------------------------------------------------------------------+
| Syntax   |  LRESFAST mc\_file                                                |
+----------+-------------------------------------------------------------------+
| Location |  ATARI\_REXT for QVME (v2.31+)                                    |
+----------+-------------------------------------------------------------------+

This command is the same as LRESPR except that it will only work on a
file in RAM disk and loads that file into FastRAM on the Atari TT.
