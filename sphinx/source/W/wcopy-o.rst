..  _wcopy-o:

WCOPY\_O
========

+----------+-------------------------------------------------------------------+
| Syntax   |  WCOPY\_O [#ch,] wild1 TO wild2                                   |
+----------+-------------------------------------------------------------------+
| Location |  THORs                                                            |
+----------+-------------------------------------------------------------------+

WCOPY\_O is the same as WCOPY\_F except that any existing files are
automatically overwritten without any prompting.

See :ref:`wcopy-f`.
