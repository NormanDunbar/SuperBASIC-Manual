HEADR
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  HEADR file, adr                                                  |
+----------+-------------------------------------------------------------------+
| Location |  TinyToolkit                                                      |
+----------+-------------------------------------------------------------------+

This command reads the file header of the specified file and stores it
at the specified address (adr) in memory. Note that file must be the
full filename.
