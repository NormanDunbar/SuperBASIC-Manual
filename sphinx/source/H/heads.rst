HEADS
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  HEADS file, adr                                                  |
+----------+-------------------------------------------------------------------+
| Location |  TinyToolkit                                                      |
+----------+-------------------------------------------------------------------+

This command sets the file header of the specified file to the 64 bytes
stored at adr.
