IDEC$
=====

+----------+-------------------------------------------------------------------+
| Syntax   |  IDEC$ (value,length,ndp)                                         |
+----------+-------------------------------------------------------------------+
| Location |  Toolkit II, THOR XVI                                             |
+----------+-------------------------------------------------------------------+

This function is exactly the same as CDEC$ except that it does not
place commas between the characters to the left of the decimal point.
