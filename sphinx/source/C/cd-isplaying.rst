CD\_ISPLAYING
=============

+----------+-------------------------------------------------------------------+
| Syntax   |  CD\_ISPLAYING                                                    |
+----------+-------------------------------------------------------------------+
| Location |  SMSQ/E for QPC                                                   |
+----------+-------------------------------------------------------------------+

This function will return 1 (True) if an Audio CD is currently playing,
otherwise it will return 0.

**CROSS-REFERENCE**

:ref:`cd-play` allows you to play an Audio CD.
