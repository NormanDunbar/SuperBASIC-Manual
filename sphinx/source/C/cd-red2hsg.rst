CD\_RED2HSG
===========

+----------+-------------------------------------------------------------------+
| Syntax   |  CD\_RED2HSG (address)                                            |
+----------+-------------------------------------------------------------------+
| Location |  SMSQ/E for QPC                                                   |
+----------+-------------------------------------------------------------------+

This function converts a specified address in HSG format into REDBOOK
format.
