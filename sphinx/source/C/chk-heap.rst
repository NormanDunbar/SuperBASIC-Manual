CHK\_HEAP
=========

+----------+-------------------------------------------------------------------+
| Syntax   |  CHK\_HEAP                                                        |
+----------+-------------------------------------------------------------------+
| Location |  SMSQ/E                                                           |
+----------+-------------------------------------------------------------------+

This command is used to check whether the heap has become corrupted - we
have no real details over its working as it is undocumented.
