COTH
====

+----------+-------------------------------------------------------------------+
| Syntax   |  COTH (x)                                                         |
+----------+-------------------------------------------------------------------+
| Location |  Hyper                                                            |
+----------+-------------------------------------------------------------------+

This function returns the hyperbolic co-tangent.

This is defined as one divided by the hyperbolic tangent, so COTH(x) =
1/TANH(x).

**CROSS-REFERENCE**

:ref:`arcoth` is the inverse function of
:ref:`coth`.
