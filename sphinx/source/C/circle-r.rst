CIRCLE\_R
=========

+----------+---------------------------------------------------------------------------------------------------------------------------------------------------+
| Syntax   | CIRCLE\_R [#ch,] x,y,radius [,ratio,ecc] :sup:`\*`\ [;x\ :sup:`i`\ ,y\ :sup:`i`\ ,radius\ :sup:`i` [,ratio\ :sup:`i`\ ,ecc\ :sup:`i`]]\ :sup:`\*` |
+----------+---------------------------------------------------------------------------------------------------------------------------------------------------+
| Location | QL ROM                                                                                                                                            |
+----------+---------------------------------------------------------------------------------------------------------------------------------------------------+

This command draws a circle relative to the current graphics cursor. See
CIRCLE.
