CD\_ISPAUSED
============

+----------+-------------------------------------------------------------------+
| Syntax   |  CD\_ISPAUSED                                                     |
+----------+-------------------------------------------------------------------+
| Location |  SMSQ/E for QPC                                                   |
+----------+-------------------------------------------------------------------+

This function will return 1 (True) if the CD is paused (as opposed to
stopped), otherwise it will return 0.

**CROSS-REFERENCE**

:ref:`cd-stop` can be used to pause the CD.

:ref:`cd-resume` resumes playing a CD.
