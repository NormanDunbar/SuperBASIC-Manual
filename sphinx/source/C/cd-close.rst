..  _cd-close:

CD\_CLOSE
=========

+----------+-------------------------------------------------------------------+
| Syntax   |  CD\_CLOSE                                                        |
+----------+-------------------------------------------------------------------+
| Location |  SMSQ/E for QPC                                                   |
+----------+-------------------------------------------------------------------+

This command closes the CD drive drawer, loading a CD if you have placed
one in the drawer.

**CROSS-REFERENCE**

:ref:`cd-eject` opens the drawer.

:ref:`cd-play` allows you to play a CD.

See :ref:`cd-init`.
