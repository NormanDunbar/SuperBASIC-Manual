CD\_TRACK
=========

+----------+-------------------------------------------------------------------+
| Syntax   |  CD\_TRACK                                                        |
+----------+-------------------------------------------------------------------+
| Location |  SMSQ/E for QPC                                                   |
+----------+-------------------------------------------------------------------+

This function returns the track number of which track on a CD is
actually being played at present.

**CROSS-REFERENCE**

:ref:`cd-play` plays specified tracks.
