.. _card-renf:

CARD\_RENF
==========

+----------+-------------------------------------------------------------------+
| Syntax   | CARD\_RENF card,old$,new$                                         |
+----------+-------------------------------------------------------------------+
| Location | SMSQ/E for Q68                                                    |
+----------+-------------------------------------------------------------------+

Renames a file in the first 16 entries of the FAT32 partition of an SD card.
