CD\_ISINSERTED
==============

+----------+-------------------------------------------------------------------+
| Syntax   |  CD\_ISINSERTED                                                   |
+----------+-------------------------------------------------------------------+
| Location |  SMSQ/E for QPC                                                   |
+----------+-------------------------------------------------------------------+

This function will return 1 (True) if there is a CD in the CD-ROM drive
and the drawer is closed, otherwise it will return 0.
