CARD\_INIT
==========

+----------+-------------------------------------------------------------------+
| Syntax   |  CARD\_INIT card\_number                                          |
+----------+-------------------------------------------------------------------+
| Location |  SMSQ/E for Q68                                                   |
+----------+-------------------------------------------------------------------+

This command initializes an SD card reader. By default only card 1 is
active on boot.
